Intro to R

This course provides an introduction to the fundamentals of the R language and its applications to data analysis.

Advanced Topics in R

A workshop on Bayesian Analysis with R-Stan, report generation with RMarkdown, and building interactive apps using RShiny.

Predictive Modeling with Python

Learn about the practice of predictive modeling using Python.

Advanced Predictive Modeling with Python

Delve further into predictive modeling with this half-day follow-up course to our original PMwP class.

Intro to Linux

This course is for researchers who have never used Linux and/or a compute cluster and introduces concepts and best practices for both.

Intro to Julia

This workshop aims to introduce users of scripting langauges and advanced programmers to the Julia ecosystem.

NGS Data Analysis

NGS workflow, general data analysis pipeline, data formats, short read alignment software, general workflow for DNA-seq, RNA-seq and ChIP-seq and corresponding software resources. Data visualization tools such as IGV.

Working with Big Data in Linux

In this Intro to Linux sequel, learn how to effectively use Linux when handling large datasets.
